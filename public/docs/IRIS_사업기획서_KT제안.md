## 2. 사업 배경 및 문제 정의

### 2-1. 내부자 유출 — 기업이 간과하는 최대 보안 위협

대한민국의 기술 유출 피해는 연간 **약 25조 원**에 달하며, 이 중 **88.3%가 내부자에 의한 유출**이다(국가정보원, 2018~2023).

삼성전자, SK하이닉스, 금융기관 등 보안 체계가 강력한 대기업도 외부 해킹은 막았지만, **내부자가 기밀을 종이에 적거나, 사진을 찍거나, USB로 복사하거나, 퇴직 시 반출하는 물리적 유출**에는 지속적으로 취약하다.

### 2-2. 2025년 — 내부자 유출이 국가적 이슈로 부상한 해

| 사건 | 시기 | 피해 규모 | 유출 수법 |
|------|------|----------|----------|
| **삼성전자 D램 공정 → 중국 CXMT 유출** | 2025.12 기소 | **최소 수십조 원** | 보안구역에서 **수백 단계 공정 레시피를 손으로 필기하여 반출**. 5년간 1.6조 원 투자한 핵심 기술 |
| **SK하이닉스 HKMG 공정 → 중국 유출** | 2025.6 대법원 확정 | 수천억 원 | 협력업체 부사장이 접근 권한을 악용하여 기술 도면/레시피 반출 |
| **쿠팡 회원 정보 유출** | 2025.11 | 3,370만 계정 접근 | 퇴직한 중국인 직원이 **5개월간 잔존 권한으로 해외 서버 통해 접근**. 고객 민원으로 비로소 발각 |
| **SK텔레콤 유심 유출** | 2025.4 | 2,695만 건 | 유심 인증키 미암호화 상태에서 해킹. 통합보안센터 신설 계기 |
| **삼성디스플레이 OLED 반복 유출** | 2020~2025 | 약 3,000억 원+ | 퇴직 연구원이 이직 시 기술자료 반출, 중국 기업과 공모하여 동종 업체 설립 |
| **Google AI 칩 설계 → 중국 유출** | 2026.1 유죄 확정 | 국가안보 위협 | 엔지니어가 500건 기밀 파일을 **개인 클라우드에 업로드**하며 중국 기업 CTO로 이중 근무 |
| **Intel 기밀 18,000건 반출** | 2024 | 소송 진행 중 | 퇴직 통보 후 **개인 NAS로 대량 복사**. 보안 시스템 차단을 3일 후 우회 성공. 이후 소재 불명 |

### 2-3. 기존 보안 솔루션의 구조적 한계

현재 기업들이 운영하는 보안 솔루션(DLP, DRM, EPP, SIEM, CCTV)은 각각의 영역만 커버하며, **물리적 유출**과 **복합 시나리오**에 대한 근본적 Gap이 존재한다.

| 유출 시나리오 | 기존 DLP | 기존 UEBA | CCTV (수동 관제) | 해결 가능? |
|-------------|:--------:|:---------:|:--------------:|:---------:|
| 수기 기록(필기) 반출 | 탐지 불가 | 탐지 불가 | 경비원 육안 의존 (사실상 불가) | **미해결** |
| 사진 촬영 | 탐지 불가 | 탐지 불가 | 경비원 육안 의존 | **미해결** |
| USB 대량 복사 | 차단 가능 | 일부 탐지 | 해당 없음 | 부분 해결 |
| 퇴직 전 대량 반출 | 일부 탐지 | 일부 탐지 | 해당 없음 | 부분 해결 |
| 복수 직원 공모 반출 | 탐지 불가 | 매우 제한적 | 해당 없음 | **미해결** |
| 협력업체 권한 악용 | 제한적 | 제한적 | 제한적 | **미해결** |

> **핵심 문제**: 기존 솔루션은 "디지털 데이터"만 본다. 그러나 삼성→CXMT 사건이 증명했듯이, **가장 치명적인 유출은 종이와 카메라로 이루어진다.** 디지털 DLP만으로는 구조적으로 막을 수 없다.

## 4. IRIS 솔루션

### 4-1. 솔루션 개요

IRIS(Internal Risk Identification System)는 디지털 행위(로그/DLP) + 물리적 행위(CCTV/출입통제) + 인사 컨텍스트(HR)를 **통합 상관분석**하여, 내부자에 의한 기밀 유출을 사전에 예측하고, 실시간으로 탐지하며, 자동으로 대응하는 AI 기반 내부자 위협 관리 플랫폼이다.

### 4-2. 3대 차별화 레이어

| 레이어 | 내용 | 경쟁사 대비 |
|--------|------|-----------|
| **Layer 1: 물리적 유출 AI 탐지** | CCTV AI가 보안구역 내 필기, 사진 촬영, 외부 녹화장치 사용을 자동 감지. 비가시 워터마크로 유출 사후 추적 | **전 경쟁사 탐지 불가. IRIS만 유일하게 커버** |
| **Layer 2: 통합 상관분석** | 디지털 + 물리 + HR 3종 데이터를 교차 분석. "야간에 기밀 문서를 반복 조회한 직원이, 다음 날 보안구역에서 필기 행위를 했다" 같은 복합 패턴을 탐지 | **기존 솔루션은 디지털만 분석. 물리/HR 연동 불가** |
| **Layer 3: 공모/복합 시나리오** | 그래프 분석(SNA)으로 복수 직원 간 비정상 데이터 전달 체인 자동 식별. 구두 전달 간접 탐지 | **경쟁사 특화 기능 부재** |

### 4-3. 패키지별 수집 데이터 / 대응 위협 / 탐지 방법 상세

---

**Small Start — "법적 부담 없이 디지털 유출부터 즉시 차단"**

| 수집 데이터 | 대응하는 위협 | 탐지 방법 |
|-----------|------------|----------|
| 파일 접근 메타데이터 (파일명, 시각, 유형, 등급) | 권한 남용 조회, 퇴직 전 대량 반출 | 업무 시간 외 기밀 파일 접근, 단기간 대량 다운로드 패턴을 룰 기반으로 탐지 |
| USB/외부장치 연결 로그 (장치종류, 시리얼, 시각) | USB/외부저장장치 유출 | 비인가 장치 즉시 차단. 승인 장치도 대량 복사 시 알림 |
| 출력 메타데이터 (출력자, 시각, 매수, 문서명) | 출력물 반출 | 일일 출력량 임계치 초과 탐지. 기밀 등급 문서 출력 시 즉시 알림. 워터마크 삽입으로 사후 추적 |
| 네트워크 접속 메타데이터 (도메인, IP, 파일 크기) | 개인 메일/메신저 전송, 클라우드 업로드 | 비인가 클라우드(개인 Google Drive 등) 접속 탐지. 개인 메일 도메인으로 대용량 파일 전송 감지 |
| 소프트웨어 실행 로그 | 화면 캡처/녹화 (소프트웨어) | 캡처 프로그램(Snipping Tool, OBS 등) 실행 즉시 차단 |
| 로그인/로그아웃 기록 | 비인가 시간대 접근 | 심야/주말 접속 패턴 이상 탐지 |
| **커버 위협: 12개 중 7~8개** | **법적 요건: PIA/노사 협의 불필요** | **구축: 4~6주, 5,000원/인/월** |

**Lite — "AI 행위분석으로 오탐을 줄이고 정확도 향상"**

Small Start의 모든 데이터에 더해:

| 추가 수집 데이터 | 추가 대응 위협 | 탐지 방법 |
|-------------|------------|----------|
| AI UEBA 행위 프로파일 (사용자별 정상 패턴 학습) | 권한 남용의 미묘한 패턴 | 개인별 베이스라인 대비 이탈 감지. "평소 10건 조회하는 직원이 갑자기 200건 조회" 같은 이상행위를 AI가 자동 탐지. 룰 기반 대비 오탐 60% 감소 |
| 이메일/메신저 전송 메타데이터 (수신자 도메인, 첨부 크기, 빈도) | 개인 메일/메신저 정밀 탐지 | 내용 열람 없이 메타데이터만 분석. "경쟁사 도메인으로 첨부 전송", "평소 안 쓰던 메신저로 대용량 전송" 탐지 |
| 퇴직 예정자 플래그 (수동 지정) | 퇴직 전 대량 반출 강화 | 퇴직 예정 직원의 모니터링 정책 자동 강화. 파일 접근/복사/출력 임계치를 50% 하향 |
| **추가 커버: 기존 위협의 탐지 정확도 대폭 향상** | **법적 요건: PIA 간이 보완, 직원 고지** | **추가 구축: 2~3주, 8,000원/인/월** |

**Standard — "물리적 유출까지 AI로 탐지하는 핵심 차별화"**

Lite의 모든 데이터에 더해:

| 추가 수집 데이터 | 추가 대응 위협 | 탐지 방법 |
|-------------|------------|----------|
| CCTV AI 영상 분석 (보안구역 내 행위 패턴) | 수기 기록(필기) 반출 | CCTV AI가 보안구역 내 "펜 들고 화면 응시 + 메모 반복" 행위를 자동 감지. 동시간대 기밀문서 조회 기록과 상관분석하여 위험 스코어 산출 |
| CCTV AI 영상 분석 (촬영 제스처) | 사진 촬영 | 스마트폰 들어올림 + 화면 방향 촬영 제스처를 감지. 비가시 워터마크로 유출 사진 사후 추적 |
| CCTV AI 영상 분석 (외부장치) | 화면 녹화 (외부장치) | 소형 캠코더, 스마트워치 등 외부 녹화장치 사용 행위 감지 |
| 출입통제 데이터 (출입시각, 구역, 체류시간) | 출력물 물리적 반출 추적 | "기밀 문서 대량 출력 → 30분 후 보안구역 퇴장"을 CCTV+출입 상관분석으로 추적 |

**Full — "전사 통합 보안 체계 + 자동 대응"**

Standard의 모든 데이터에 더해:

## 5. 경쟁 환경 및 차별화

### 5-1. 경쟁사 실제 기능 비교 (물리적 유출 영역)

기존 경쟁사도 디지털 DLP/UEBA 영역에서는 상당한 기능을 보유하고 있다. IRIS의 진정한 차별화는 **경쟁사가 전혀 커버하지 못하는 물리적 유출 영역**에 있다.

| 유출 시나리오 | Proofpoint | Teramind | Microsoft Purview | 파수 | 소만사 | **IRIS** |
|-------------|:---------:|:--------:|:----------------:|:---:|:-----:|:-------:|
| 수기 기록 반출 | 불가 | 불가 | 불가 | 불가 | 불가 | **CCTV AI 자동 감지** |
| 사진 촬영 | 불가 | 불가 | 불가 | 불가 | 불가 | **CCTV AI + 비가시 워터마크** |
| 화면 녹화 (외부장치) | 불가 | SW만 감지 | 불가 | 불가 | SW만 차단 | **SW 차단 + CCTV AI** |
| 출력 후 물리 반출 추적 | 불가 | 불가 | 불가 | 불가 | 불가 | **출력 워터마크 + CCTV+출입 상관분석** |

### 5-2. 경쟁 포지셔닝

| | IRIS와의 관계 | 대응 전략 |
|--|:-----------:|----------|
| **Proofpoint** (최대 위협) | 글로벌 경쟁 | 2022년 한국 진출. 물리보안 미커버를 차별화로 대응. "Proofpoint은 디지털만, IRIS는 물리까지" |
| **Teramind** | 글로벌 간접 경쟁 | 한국 미진출. 한국어/규제 특화로 차별화 |
| **MS Purview** | 대체재 | M365 번들 무료 위력. **보완재 포지셔닝** (Purview=M365 내, IRIS=물리+비M365) |
| **파수** | **보완재/파트너** | DRM(문서 암호화) + IRIS(행위 분석) = 이상적 조합 |
| **소만사** | **보완재/파트너** | DLP(매체 제어) + IRIS(상관분석) = 번들 제안 |
| **안랩** | 보완재 | EPP(외부 위협) + IRIS(내부 위협) = 내외부 통합 |

### 5-3. 모방 난이도

| 경쟁사 유형 | 물리+디지털 통합 모방 | 한국 시장 특화 모방 | SI 번들 모방 | 종합 |
|-----------|:------------------:|:-----------------:|:-----------:|:----:|
| 글로벌 (Proofpoint, Teramind, MS) | 어려움 (물리보안 역량 없음) | 어려움~보통 | 어려움 (SI 역량 없음) | **높음** |
| 국내 (파수, 소만사, 안랩) | 보통 (각각 부분 역량만) | 쉬움 | 보통 | **중간** |

> **결론**: 글로벌 벤더의 모방에 최소 2~3년 소요. 국내 기업은 부분 역량만 보유하여 단기간 통합 솔루션 구현 어려움.
