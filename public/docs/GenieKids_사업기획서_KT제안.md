## 1. Executive Summary

**한 줄 요약**: 기가지니 400만 가입 기반에 AI 양방향 키즈 콘텐츠를 탑재하여, 집(기가지니) · 외출(핸드폰) · 차(블루투스)에서 아이와 대화하는 국내 최초 멀티 디바이스 AI 키즈 플랫폼

> 핑크퐁과 티니핑은 아이가 "보는" 콘텐츠입니다. 지니야~?는 아이가 "말하는" 콘텐츠입니다. 기가지니라는 400만 디바이스가 이미 거실에 있고, 부모의 핸드폰이 이미 주머니에 있습니다. 하드웨어 투자 없이, 소프트웨어만으로 차세대 키즈 플랫폼을 시작할 수 있습니다.

| 항목 | 내용 |
|------|------|
| **서비스명** | 지니야~? (Genie Kids) |
| **서비스 유형** | 기가지니 키즈 부가서비스 (월정액 구독) |
| **타겟 사용자** | 3~9세 자녀를 둔 부모, 조부모 (약 220만 가구) |
| **핵심 차별화** | 국내 유일 AI 양방향 음성 대화 키즈 콘텐츠. 틀어놓는 영상이 아닌, 아이와 대화하는 AI 친구 |
| **디바이스 전략** | 집(기가지니/지니TV) + 외출(KT 모바일 앱) + 차(블루투스 연결) — 끊김 없는 경험 |
| **기가지니 시너지** | 400만 기가지니 가입자에 즉시 노출. 기존 요금제에 부가 옵션으로 추가. IPTV/지니TV 940만 가입자 확장 가능 |
| **콘텐츠 전략** | 자체 캐릭터 IP(8종) + 콘텐츠 제작 협력사 3~5곳 + 기존 동요/동화 라이선스 |
| **수익 모델** | 월 4,900~9,900원 구독 + 캐릭터 MD + 교육기관 B2B + 콘텐츠 라이선싱 |
| **투자 규모** | Phase 0(Prototype): 8,000만 원 → Phase 1(MVP): 3억 → Phase 2(정식론칭): 5억. 누적 약 8.8억 원 |
| **예상 수익** | 보수적 3년차: 매출 62억(구독 20만 가구), 이상적 3년차: 매출 187억(구독 50만 가구) |
| **요청 사항** | Phase 0 착수 승인 (8,000만 원, 3개월, 4명) |

## 2. 사업 배경 및 기회

### 2-1. 변화하는 육아 환경

| 트렌드 | 현황 | 사업 기회 |
|--------|------|----------|
| **저출산이지만 프리미엄 육아** | 합계출산율 0.68(세계 최저)이지만, 자녀 1인당 교육비/콘텐츠 지출은 역대 최고 | 아이 수는 줄어도 아이당 지출은 증가. 프리미엄 서비스 기회 |
| **맞벌이 가구 증가** | 맞벌이 비율 50%+ → 육아 도우미/조부모 양육 증가 | 조부모도 쉽게 쓸 수 있는 음성 기반 서비스 수요 |
| **스크린타임 우려** | 유아 일일 스마트폰/태블릿 평균 2시간+ → 부모 죄책감 | 화면 없이 음성으로만 상호작용 = 스크린타임 대안 |
| **AI 네이티브 세대** | 2020년 이후 출생 아이들은 AI 비서가 일상인 환경에서 성장 | 아이가 AI와 자연스럽게 대화하는 서비스 수용도 높음 |

### 2-2. KT 기가지니의 미활용 기회

| 기가지니 자산 | 현황 | 지니야~? 활용 |
|-------------|------|-------------|
| **가입자 400만+** | IPTV/지니TV 940만 가입자 | 기가지니 키즈 부가서비스로 즉시 노출. 별도 앱 설치 불필요 |
| **AI 음성 인식** | KT 자체 LLM '믿음' 탑재, 음성인식률 고도화 중 | 아이 음성 인식에 특화 튜닝 (발음 부정확, 짧은 문장, 높은 톤) |
| **거실 디바이스** | 기가지니 스피커/셋톱박스가 거실에 상시 설치 | 별도 기기 구매 불필요. "지니야~?"로 즉시 호출 |
| **KT 모바일 가입자** | KT 모바일 1,700만+ 가입자 | 외출 시 KT 앱에서 동일 서비스 연속 이용 |
| **키즈 콘텐츠 부재** | 현재 기가지니에 본격적인 AI 키즈 서비스 없음 | 경쟁사(SKT NUGU, Kakao Mini) 대비 키즈 차별화 기회 |

### 2-3. 왜 KT가 해야 하는가

> 핑크퐁(유튜브)과 티니핑(TV)은 아이가 "보는" 일방향 콘텐츠입니다. 아이가 "말하는" 양방향 AI 콘텐츠를 만들 수 있는 기업은 **400만 AI 스피커를 거실에 깔아놓은 KT뿐**입니다. SKT NUGU는 가입자 열세, 카카오 미니는 사실상 중단. 이 기회는 KT에게만 열려 있습니다.

## 4. 지니야~? 서비스 개요

### 4-1. 핵심 컨셉

| | 핑크퐁/티니핑 (기존) | **지니야~? (IRIS)** |
|--|:------------------:|:------------------:|
| **상호작용** | 일방향 (보기만) | **양방향 (AI와 대화)** |
| **디바이스** | 스마트폰/태블릿/TV | **기가지니 + 핸드폰 + 차(블루투스)** |
| **개인화** | 없음 (모든 아이에게 같은 콘텐츠) | **AI가 아이 수준/관심사/성장 단계에 맞춤** |
| **성장 추적** | 불가 | **주간 성장 리포트 (언어/수리/창의/사회성)** |
| **부모 참여** | 틀어주기만 | **부모앱에서 리포트 확인 + 콘텐츠 설정** |
| **스크린타임** | 화면 의존 | **음성 기반 = 스크린 프리 옵션** |

### 4-2. 콘텐츠 구성 (론칭 시)

| 카테고리 | 콘텐츠 수 | 주요 내용 | 상호작용 방식 |
|---------|:--------:|----------|:----------:|
| 동요/음악 | 150곡 | 한국 동요 + 영어 동요 + 지니 오리지널 | 같이 노래, 가사 퀴즈, 리듬 게임 |
| 동화/스토리 | 300편 | 전래동화 + 세계명작 + 지니 오리지널 | 인터랙티브 스토리 (선택지 분기) |
| 놀이/게임 | 100개 | 끝말잇기, 스무고개, 퀴즈, 수수께끼 | 완전 양방향 음성 게임 |
| 학습 | 300개 | 한글/영어/수학/과학 | AI 튜터 (수준별 맞춤, 피드백) |
| 수면/자장가 | 60곡 | 자장가 + 백색소음 + 수면 이야기 | 잠들면 자동 종료 (소리 감지) |
| 역할놀이 | 50개 | 소방관, 우주비행사, 요리사, 수의사 | AI 캐릭터와 역할극 대화 |
| **합계** | **960개** | | |

### 4-3. AI 캐릭터 8종

| 캐릭터 | 역할 | 성격 | 대표 대사 |
|--------|------|------|----------|
| **지니** | 메인 AI 친구 | 밝고 다정, 호기심 많음 | "응! 부를 때마다 설레! 오늘은 뭐 하고 놀까?" |
| **별이** | 음악 담당 | 활발, 리듬감 넘침 | "랄랄라~ 어떤 노래가 듣고 싶어?" |
| **달이** | 수면/자장가 | 차분하고 포근함 | "오늘도 수고했어~ 포근한 자장가 들려줄게..." |
| **해나** | 학습 도우미 | 똑똑하고 인내심 강함 | "좋아! 오늘은 뭘 배워볼까? 퀴즈도 준비했어!" |
| **구름이** | 동화 스토리텔러 | 상상력 풍부 | "옛날 옛적에~ 아주 먼 나라에..." |
| **무지** | 놀이/게임 | 장난꾸러기 | "신난다! 오늘은 어떤 게임 할까?" |
| **나라** | 자연/과학 | 차분하고 지혜로움 | "공룡은 아주 오래전에 살았어~" |
| **하늘이** | 감정/사회성 | 공감 능력 뛰어남 | "괜찮아, 이야기해줄래? 내가 들어줄게~" |

> 8개 캐릭터는 KT 자체 IP로 개발. 향후 캐릭터 상품(인형, 문구, 의류), 애니메이션, 테마파크 등 IP 비즈니스로 확장.

## 5. 멀티 디바이스 전략

### 5-1. "끊김 없는 경험" — 집, 외출, 차

| 환경 | 디바이스 | 사용 시나리오 | 기술 구현 |
|------|---------|-------------|----------|
| **집 (거실/아이방)** | 기가지니 스피커, 지니TV 셋톱박스 | "지니야, 동화 들려줘" — 거실에서 음성으로 호출. 지니TV 화면에 캐릭터 애니메이션 동시 표시 | 기가지니 SDK 활용. 기존 기가지니 앱에 키즈 모드 추가 |
| **외출 (공원/마트/병원)** | 부모 핸드폰 (KT 모바일 앱) | 외출 시 앱 실행 → 이어폰으로 "지니야, 심심해" → 지니가 퀴즈/이야기/노래 제공. 대기 시간 활용 | KT 기가지니 모바일 앱 내 키즈 탭 추가 |
| **차 (이동 중)** | 핸드폰 → 차량 블루투스 | 장거리 이동 시 블루투스로 차량 스피커 연결 → "지니야, 노래 틀어줘" → 가족 함께 듣기. 화면 없이 음성으로만 | 표준 블루투스 A2DP. 별도 차량 개발 불필요 |
| **조부모 댁** | 조부모 핸드폰/기가지니 | 손주가 조부모 댁 방문 시 조부모 기가지니에서 동일 서비스. "지니야, OO이가 좋아하는 노래 틀어줘" → 아이 프로필 동기화 | 가족 계정 공유. 프로필 클라우드 동기화 |

### 5-2. 조부모 친화 설계

맞벌이 가구에서 조부모가 양육하는 시간이 증가하고 있으며, 기가지니 AI 스피커 사용자 중 신중년이 37.8%로 1위입니다.

| 설계 원칙 | 구현 |
|----------|------|
| **음성 전용 인터페이스** | 앱 조작 불필요. "지니야, 아이 동요 틀어줘"만으로 시작 |
| **자동 콘텐츠 추천** | 아이 연령/시간대에 맞는 콘텐츠 자동 재생 (오전=학습, 오후=놀이, 저녁=자장가) |
| **간편 설정** | 부모가 모바일 앱에서 원격으로 콘텐츠/시간 설정. 조부모는 "지니야" 호출만 |
| **이용시간 자동 제한** | 부모가 설정한 일일 이용시간(30분/60분/90분) 초과 시 자동 종료 |
| **성장 리포트 공유** | 조부모 댁에서의 이용 내역도 부모 앱에 자동 리포트 |
